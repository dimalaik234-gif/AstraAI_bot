# Развёртывание на Bothost.ru

## Пошаговая инструкция

### 1. Подготовка токенов

1. **Telegram Bot Token**
   - Открой @BotFather в Telegram
   - Отправь `/newbot`
   - Придумай имя и username
   - Скопируй токен (выглядит как `123456789:AAF...`)

2. **OpenRouter API Key (бесплатно)**
   - Перейди на [https://openrouter.ai](https://openrouter.ai)
   - Зарегистрируйся (можно через Google/GitHub)
   - Перейди в раздел **API Keys**
   - Нажми **Create Key**
   - Скопируй ключ (начинается с `sk-or-...`)

### 2. Загрузка на Bothost.ru

#### Вариант A: Через Git (рекомендуется)

1. Создай репозиторий на GitHub (или GitLab)
2. Загрузи туда файлы проекта:
   - `bot.py`
   - `requirements.txt`
   - `README.md`
   - `.env.example` (можно переименовать в `.env` и заполнить локально, но **НЕ** коммить `.env`!)
3. Зарегистрируйся на [bothost.ru](https://bothost.ru)
4. Создай новый проект → выбери **Telegram Bot**
5. Подключи Git-репозиторий
6. В настройках проекта добавь **переменные окружения**:
   - `BOT_TOKEN` = твой токен из BotFather
   - `OPENROUTER_API_KEY` = твой ключ из OpenRouter
   - `DEFAULT_MODEL` = `deepseek/deepseek-chat-v3-0324:free` (или другой)
7. Укажи **Entrypoint / Main file**: `bot.py`
8. Нажми **Deploy** / Развернуть

#### Вариант B: Загрузка файлов вручную

1. Создай проект на Bothost
2. Перейди в раздел **Файлы**
3. Загрузи все файлы проекта
4. Добавь переменные окружения (как выше)
5. Укажи entrypoint `bot.py`

### 3. После деплоя

- Bothost автоматически выдаст публичный URL
- Бот должен сразу заработать (использует polling)
- Если хочешь Webhook — можно настроить позже (Bothost поддерживает)

### 4. Проверка

Напиши боту в Telegram:
- `/start`
- Просто любое сообщение

### 5. Важно

- Бесплатный тариф Bothost имеет ограничения по RAM/CPU. Для простого ИИ-бота хватает.
- OpenRouter бесплатные модели имеют rate limits. При превышении бот сообщит.
- История чата хранится в оперативной памяти — при перезапуске сбрасывается.

### Полезные команды после деплоя

- `/current` — посмотреть текущие настройки
- `/modes` — переключить режим
- `/model gemini` — сменить модель

### Обновление кода

При использовании Git:
- Просто запушь изменения на GitHub
- Bothost автоматически подхватит (или нажми Redeploy)

Готово! Бот должен работать.
